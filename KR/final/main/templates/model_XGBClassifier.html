XGB Classifier



XGB(Extreme Gradient Boosting) Classifier 정의

- 앙상블 모델의 한 종류인 boosting의 종류이다. 부스팅은 약한 분류기를 세트로 묶어서 정확도를 예측하는 기법이다.
- Xgboosting은 gradient boosting 알고리즘의 단점을 보완해 주기 위해 나왔다.
  (gradient boosting 의 단점 - 느리다, 과적합 이슈)
- xgboost의 특징
1. gbm보다 빠르다
2. 자동 가지치기를 통해 과적합이 잘 일어나지 않는다.
3. 다른 알고리즘과 연계 활용성이 좋다.
4. 다양한 커스텀 최적화 옵션을 제공한다. 유연성이 좋다.
